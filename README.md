# Text Mining and Social Media Mining

This repository presents an academic text mining project focused on consumer review analysis in Amazon’s **All Beauty** category. The project investigates how customer language varies across satisfaction levels, which themes dominate positive and negative reviews, and which linguistic features are associated with perceived review helpfulness and verified purchases.

## Project Overview
The analysis applies a combination of text mining and machine learning techniques to large-scale online review data, including:
- TF-IDF n-grams
- Log-odds characteristic term analysis
- Sentiment analysis
- Topic modeling (LDA)
- Clustering and semantic structure exploration
- Helpfulness and engagement modeling

The project is designed to answer three main research questions:
1. How does review language differ across rating levels?
2. What themes are common in positive and negative reviews?
3. Which linguistic features are associated with helpfulness and verified purchase signals?

This project highlights practical applications of:

## Purpose
The purpose of this repository is to showcase applied analytical work in text mining and social media mining, with an emphasis on reproducible research, structured methodology, and interpretable results.
